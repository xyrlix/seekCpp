# C++ 网络编程训练

## 概述
网络编程是 C++ 的重要应用领域。本 Skill 从基础 Socket 到高性能网络框架逐步训练。

## 训练模块

### Module 1: Socket 基础（第 1-3 天）
- TCP/UDP 基础
- Socket API
- 阻塞 vs 非阻塞
- 地址结构

### Module 2: 多路复用（第 4-6 天）
- select/poll
- epoll (Linux)
- IOCP (Windows)
- Reactor 模式

### Module 3: 协议设计（第 7-8 天）
- 消息格式
- 序列化
- 粘包处理
- 心跳机制

### Module 4: 高性能网络（第 9-12 天）
- 线程模型
- 零拷贝
- 连接池
- 负载均衡

## 练习项目
- 回声服务器
- HTTP 服务器
- RPC 框架
- 聊天室

## 达标标准
- 能实现 TCP/UDP 通信
- 掌握多路复用
- 能设计网络协议
- 理解高性能架构
