# C++ 系统工程训练

## 概述
系统工程能力是专家级开发者的标志。本 Skill 训练构建系统、链接模型、跨平台开发、CI/CD 流程。

## 训练模块

### Module 1: 构建系统（第 1-3 天）
- CMake 高级用法
- 目标属性配置
- 依赖管理
- 自定义命令

### Module 2: 链接模型（第 4-5 天）
- 静态链接
- 动态链接
- 符号可见性
- ODR 规则

### Module 3: ABI 兼容（第 6-7 天）
- ABI 稳定性
- 版本管理
- 跨编译器兼容
- PIMPL 惯用法

### Module 4: 跨平台开发（第 8-10 天）
- 平台检测
- 条件编译
- 路径处理
- 打包分发

### Module 5: 测试与 CI（第 11-14 天）
- Google Test 框架
- 测试覆盖率
- CI/CD 配置
- 自动化发布

## 练习项目
- CMake 模块化项目
- 动态库封装
- 跨平台工具链
- GitHub Actions 配置

## 达标标准
- 能配置复杂构建系统
- 理解链接模型
- 掌握跨平台开发
- 能搭建 CI/CD 流程
