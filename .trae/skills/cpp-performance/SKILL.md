# C++ 性能优化训练

## 概述
性能优化是专家级 C++ 开发者的核心能力。本技能通过系统训练让学习者掌握性能分析、优化策略和最佳实践。

## 训练模块

### Module 1: 性能分析工具（第 1-2 天）
- chrono 计时
- 性能分析器（perf, VTune）
- 内存分析工具
- 火焰图生成

### Module 2: 基准测试（第 3-4 天）
- Google Benchmark 使用
- 编写有意义的基准
- 避免基准测试陷阱
- 统计分析方法

### Module 3: 缓存优化（第 5-6 天）
- 缓存局部性原理
- 数据结构布局优化
- 伪共享问题
- 预取优化

### Module 4: 内存优化（第 7-8 天）
- 自定义分配器
- 内存池设计
- 零拷贝技术
- 对象重用

### Module 5: 编译优化（第 9-10 天）
- 编译器优化级别
- 内联策略
- 循环优化
- SIMD 指令

## 练习项目
- 矩阵乘法优化
- 字符串处理优化
- 容器性能对比
- 内存分配器实现

## 达标标准
- 能使用性能分析工具
- 能识别性能瓶颈
- 掌握常见优化策略
- 能编写性能测试
