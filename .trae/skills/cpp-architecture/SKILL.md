# C++ 架构设计训练

## 概述
架构设计能力决定能否从"写代码"升级到"设计系统"。本 Skill 训练接口设计、依赖管理、组件化架构。

## 训练模块

### Module 1: 接口设计（第 1-3 天）
- 接口隔离原则
- 抽象基类设计
- 接口版本管理
- 回调与事件

### Module 2: 依赖管理（第 4-5 天）
- 依赖注入
- 控制反转
- 组件解耦
- 插件架构

### Module 3: 错误处理（第 6-8 天）
- 错误码设计
- 异常策略
- std::expected (C++23)
- 错误传播

### Module 4: 设计模式（第 9-12 天）
- 工厂模式
- 观察者模式
- 策略模式
- 访问者模式
- CRTP 模式

### Module 5: 组件化（第 13-14 天）
- 模块边界
- 通信机制
- 生命周期管理
- 配置系统

## 练习项目
- 插件系统
- 事件总线
- 配置管理器
- 日志框架

## 达标标准
- 能设计清晰的接口边界
- 掌握依赖管理
- 能选择合适的设计模式
- 理解架构权衡
