# C++ 基础语法训练系统

## 概述

本 Skill 为 C++ 学习者提供系统化的基础语法训练，涵盖变量、控制流、函数、指针、引用、数组等核心概念。通过大量刻意练习建立可靠的编程基础。

## 训练阶段

### Phase 1.1: 变量与数据类型（第 1-3 天）
- 基本数据类型：int, float, double, char, bool
- 变量声明与初始化
- const 修饰符与常量
- 运算符：算术、逻辑、位运算

**每日任务：**
- 手写 5-10 个小例子
- 完成 10 道递进练习题
- AI 检查代码质量

### Phase 1.2: 控制流（第 4-6 天）
- if-else 条件判断
- switch-case 语句
- for/while/do-while 循环
- break/continue/return

**每日任务：**
- 实现 3 个控制流小程序
- 完成 8 道循环嵌套题
- 检查边界条件和死循环风险

### Phase 1.3: 函数基础（第 7-9 天）
- 函数声明与定义
- 参数传递：值传递、引用传递
- 返回值与 return 语句
- 函数重载
- 默认参数

**每日任务：**
- 实现 5 个函数示例
- 完成 7 道函数设计题
- 检查参数验证和返回值处理

### Phase 1.4: 数组与字符串（第 10-12 天）
- 一维数组与多维数组
- 字符数组与 C 风格字符串
- std::string 基础使用
- 数组与指针的关系

**每日任务：**
- 实现 4 个数组操作程序
- 完成 6 道字符串处理题
- 检查越界和内存安全问题

### Phase 1.5: 指针与引用（第 13-16 天）
- 指针基础与解引用
- 空指针与野指针
- 指针算术运算
- 引用与指针的区别
- 指针作为函数参数

**每日任务：**
- 实现 6 个指针操作示例
- 完成 10 道指针练习题
- 检查悬空指针和内存泄漏

## 训练方法

### 1. 刻意练习原则
- 目标明确：每次训练聚焦 1 个小主题
- 难度略高：超出舒适区但可完成
- 立即反馈：AI 检查每段代码
- 持续重复：错题重做直到掌握

### 2. 错误库管理
每次训练后：
- 记录所有错误和疑问
- 分类错误类型（语法、逻辑、内存、设计）
- 总结错误原因和正确做法
- 下周复习时重做错题

### 3. AI 反馈提示词
使用以下固定提示词获取高质量反馈：

```
"请检查这段代码是否存在：
1. 语法错误
2. 越界访问
3. 悬空引用/指针
4. 未初始化变量
5. 资源泄漏
6. 逻辑错误"
```

```
"请用初学者能懂的方式解释这个 C++ 概念，
再给出一个容易犯错的例子。"
```

## 达标标准

完成本阶段后应能：
- 独立编写无资源泄漏的中小程序
- 正确理解指针和引用的区别
- 避免常见越界和未初始化问题
- 解释变量生命周期和作用域
