# C++ 专家级训练 12 周计划

## 训练原则

- **高密度刻意练习**：每天 2-3 小时专注编码
- **项目驱动**：每 2 周产出 1 个可运行项目
- **持续反馈**：AI 代码审查 + 错题复盘
- **渐进难度**：每周难度提升 20%
